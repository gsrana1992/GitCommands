Git Commands:

Create folder/directory to be maintained : mkdir project
move into the folder/directory : cd project/
Initializing Git Repository: git init
Making the first change : touch names.txt
checking status after last commit and for any changes made : git status
Staging the changes : git add names.txt
Commiting the changes : git commit -m "Add names.txt file"
Viewing the overall history of the project : git log
Removing changes from stage : git restore --staged names.txt
Restoring any deleted file/folder : git restore names.txt
Removing a commit from the history of a project : git reset a84b22d73f07a7bd4c028643d2bbcd8aefc1164a
Stashing changes : git stash
Popping Stash : git stash pop
Clearing Stash : git stash clear
Connecting Remote Repository to Local Repository : git remote add origin https://github.com/gsrana1992/GitCommands.git
Pushing local changes to remote repository : git push origin master
Create new branch : git branch branch_cmd
Move head to the new branch : git checkout branch_cmd
Merge 2 branches : 1. move to the desired branch => git checkout master ; 2. merge the branch => git merge branch_cmd
Clone a repository : git clone https://github.com/gsrana1992/disney-data-science-tasks.git
add upstream to your git repository : git remote add upstream https://github.com/KeithGalli/disney-data-science-tasks.git
Push to remote after resetting to older commit : git push origin master -f
sync your main branch with upstream main branch : git fetch --all --prune ; git reset --hard upstream/main **OR** git pull upstream master
squash multiple commits in one commit : git rebase -i "header previous to the oldest header that needs to be squashed"
